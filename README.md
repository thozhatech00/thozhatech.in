# Thozha Tech Website V2

This is the simplified root-only upload package for GitHub Pages.

## Upload these files to the repository root

- `index.html`
- `404.html`
- `CNAME`
- `.nojekyll`
- `README.md`

No assets folder is required in this version. Logo, CSS, and JavaScript are embedded inside `index.html`.

## GitHub Pages Settings

Go to:

`Settings -> Pages`

Use:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ root`
- Custom domain: `thozhatech.in`
