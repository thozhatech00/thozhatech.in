# Thozha Tech Professional Website

Premium static website for **Thozha Tech**.

## Files

- `index.html` — complete website page
- `style.css` — premium responsive styling
- `script.js` — mobile menu, animation and footer year
- `assets/logo.svg` — full logo
- `assets/logo-mark.svg` — icon logo
- `assets/favicon.svg` — browser icon
- `CNAME` — GitHub Pages custom domain

## GitHub Pages Deployment

1. Extract this zip.
2. Upload/replace files in the GitHub repository root.
3. Keep the `CNAME` file as `thozhatech.in`.
4. Commit changes to the `main` branch.
5. Open `https://thozhatech.in`.
