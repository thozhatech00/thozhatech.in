# Thozha Tech Website V3

Premium static website package for GitHub Pages.

## Upload to GitHub repository root

Upload these files directly in the repository root, not inside a folder:

- index.html
- 404.html
- CNAME
- .nojekyll
- logo.svg
- logo-mark.svg
- favicon.svg
- README.md

## GitHub Pages settings

- Source: Deploy from a branch
- Branch: main
- Folder: / root
- Custom domain: thozhatech.in
- Enforce HTTPS: enabled
